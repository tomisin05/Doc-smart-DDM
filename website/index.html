About Doc-smart

Doc-smart was created to solve the common problem debaters face: managing and quickly accessing numerous Word documents during preparation and competition. Built with Python and Tkinter, it provides a lightweight yet powerful solution for debate document management.

Technical Specifications

  • Built with Python 3.x
  • Tkinter GUI framework
  • Windows COM integration for Word
  • JSON data storage
  • Cross-platform compatible
